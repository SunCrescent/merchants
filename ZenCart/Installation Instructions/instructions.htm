Step 1) Connect to your domain via FTP or similar file transfer method.

Step 2) Copy the file "cavirtex_callback.php" in "Files to upload" to the root of your zencart site (e.g. to /var/www/html/zencart/cavirtex_callback.php ).

Step 3) Copy the file "cavirtex.php" in "Files to upload/languages" to the zencart payment module languages folder.

(e.g. to " /var/www/html/zencart/includes/languages/english/modules/payment/cavirtex.php ").

Step 4) Copy the file "cavirtex.php" in "Files to upload/modules" to the zencart payment modules folder.

(e.g. to " /var/www/html/zencart/includes/modules/payment/cavirtex.php ").

Step 5) Log in to the admin section of your zencart store.

Go to "Modules" and select "Payment". Then, locate "Payment with Bitcoins" in the list of modules and select it. Finally, click "Install".

Step 6) Enter appropriate variable details.

The CaVirtEx Merchant Key and CaVirtEx Merchant Secret Key can be found on the merchant section of the cavirtex.com website.

The first Set Order Status should be set to what unpaid orders should be (e.g. pending).

The second Set Order Status should be set to what paid orders should be (e.g. processing).

Click "update" when you are done configuring the module.

Step 7) On the CaVirtEx site, click "Merchant Profile". Find IPN URL, and enter the URL for your ZenCart's cavirtex_callback.php file. Click "Update" when done.

e.g. http://aaronk.ca/zencart/cavirtex_callback.php

You can now accept payments with bitcoins!
