Step 1) Connect to your domain via FTP or similar file transfer method.

Step 2) Copy the file "cavirtex_callback.php" in "Files to upload" to the root of your osCommerce site (e.g. to /var/www/html/oscom/cavirtex_callback.php ).

Step 3) Copy the file "cavirtex.php" in "Files to upload/languages" to the osCommerce payment module languages folder.

(e.g. to " /var/www/html/oscom/includes/languages/english/modules/payment/cavirtex.php ").

Step 4) Copy the file "cavirtex.php" in "Files to upload/modules" to the osCommerce payment modules folder.

(e.g. to " /var/www/html/oscom/includes/modules/payment/cavirtex.php ").

Step 5) Log in to the admin section of your osCommerce store.

Go to "Modules" and select "Payment".

Step 6) Click on the "Install Module" button.

Step 7) Locate "Payment with Bitcoins" in the list of modules and select it. Finally, click "Install Module".

Step 8) With "Payment with Bitcoins" still selected, click the "Edit" button.

Step 9) Enter appropriate variable details.

The CaVirtEx Merchant Key and CaVirtEx Merchant Secret Key can be found on the merchant section of the cavirtex.com website.

The Set Order Status (unpaid) option should be set to what unpaid orders should be changed to (e.g. pending).

The Set Order Status (paid) option should be set to what paid orders should be changed to (e.g. processing).

Click "save" when you are done configuring the module.

Step 10) On the CaVirtEx site, click "Merchant Profile". Find IPN URL, and enter the URL for your osCommerce's cavirtex_callback.php file. Click "Update" when done.

e.g. http://aaronk.ca/oscom/cavirtex_callback.php

You can now accept payments with bitcoins!
