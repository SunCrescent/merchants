Prestashop Bitcoin payment module

This is prestashop payment module that uses cavirtex.com as payment gateway for shop transactions. 

How to install:
---------------
1. Using your FTP client, upload the "virtex" folder to /modules/ folder on your server
2. Login to admin panel of your shop and goto Modules section, in the list you'll find " 	
Virtex payment gateway" listed. Click on "Install" to install the module. After module has been installed, click on Configure link to edit your module. In module configuration you need to enter your API Key that you got by registering on http://cavirtex.com, after you enter you API Key, click "Save settings". Once API has been saved you will be able to see your payment module in the list on the front page during payment process
